# E-Commerce Platform Requirements

## User Management
- Users can register with email
- Users can login
- Users can reset password

## Product Catalog
- Display products with images
- Search products by name
- Filter by category
- Sort by price

## Shopping Cart
- Add items to cart
- Update quantities
- Remove items
- Calculate total

## Payment
- Support credit card payment
- Support PayPal
- Generate invoice after payment
